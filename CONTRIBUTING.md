# Contributing to DynamicNotch

Thanks for your interest in contributing! 🎉

## How to Contribute

### Report Bugs

1. Check [existing issues](https://github.com/MedSageYu/DynamicNotch/issues) first
2. Open a new issue with:
   - macOS version and Mac model
   - Steps to reproduce
   - Expected vs actual behavior
   - Screenshots if applicable

### Suggest Features

Open an issue with the **feature request** label. Describe:
- The problem you're trying to solve
- Your proposed solution
- Any alternatives you considered

### Submit Code

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test on macOS 14+ (Apple Silicon preferred)
5. Commit: `git commit -m 'Add your feature'`
6. Push: `git push origin feature/your-feature`
7. Open a Pull Request

### Code Style

- Swift 6.3+ with SwiftUI
- Follow existing code patterns
- Keep functions focused and small
- Add comments for non-obvious logic

## Development Setup

```bash
git clone https://github.com/MedSageYu/DynamicNotch.git
cd DynamicNotch
swift build
open ~/Applications/DynamicNotch.app
```

## Questions?

Open an issue or start a discussion. We're friendly! 😊
