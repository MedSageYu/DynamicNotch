# 💊 Pill

A sleek pill-shaped command center for your Mac.

## Download

**[Download Pill.zip](https://github.com/MedSageYu/Pill/releases/download/v1.0.0/Pill.zip)**

1. Download `Pill.zip`
2. Unzip
3. Double-click `Pill.app` to run

> **First time?** If macOS says "cannot be opened because the developer cannot be verified":
> - Right-click `Pill.app` → select **Open**
> - Click **Open** in the dialog
> - It will work normally after that

No terminal. No Xcode. No extra software.

## What it does

| Feature | Description |
|---------|-------------|
| 🎵 Music | Play/pause/skip with album art |
| 🪞 Mirror | Camera preview |
| 📅 Calendar | 7-day view with events |
| 📋 Clipboard | Last 20 copied items |
| 📡 AirDrop | Drag files to share |
| ⚙️ Settings | Customize appearance and behavior |

## Requirements

- macOS 14.0 or later
- Apple Silicon Mac (M1/M2/M3/M4)

## How it works

Hover at the top center of your screen to expand. Click to interact. Move away to collapse.

## Permissions

The app will ask for:
- **Camera** — for the mirror feature
- **Calendar** — to show your events
