# 💊 Pill — Your Mac's New Best Friend

**A sleek, pill-shaped command center that lives at the top of your screen.**

One hover. One click. Everything you need.

---

## ✨ What's Inside

### 🎵 Music — Now Playing, Beautifully
See what's playing right now. Album art, track name, artist. Play, pause, skip — all from your menu bar. No need to switch apps.

### 🪞 Mirror — Instant Selfie
Need a quick camera check? One tap and you're looking at yourself. Flip horizontally if you want. Perfect for video calls.

### 📅 Calendar — Your Week at a Glance
7 days, scrollable, with today highlighted. See your upcoming events without opening Calendar.app. Filter by category — work, personal, travel — you choose what shows up.

### 📋 Clipboard — Never Lose Anything
Automatically saves your last 20 copies — text and images. Horizontal card layout (like a social media feed). Hover to see more. One click to copy back. Auto-cleans old items.

### 📡 AirDrop — Drag, Drop, Send
Drag a file near the pill and it expands automatically. Drop it in, pick a device, done. Includes a file tray for temporary storage with smart cleanup policies.

### ⚙️ Settings — Make It Yours
- Auto-collapse timer (0.5–10 seconds)
- Pill height adjustment
- Camera mirror flip
- Calendar category filters
- File tray cleanup rules

## 🛠️ Tech Stack

| Component | Technology |
|-----------|-----------|
| UI Framework | SwiftUI + AppKit |
| Music Control | AppleScript (Apple Music) |
| Calendar | EventKit (EKEventStore) |
| Camera | AVFoundation (AVCaptureSession) |
| Clipboard | NSPasteboard polling |
| AirDrop | NSSharingService |
| Mouse Tracking | Global EventMonitor (NSEvent) |
| Build System | Swift Package Manager |

**No Xcode required.** Just `swift build` and go.

## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/MedSageYu/Pill.git
cd Pill

# Build
swift build

# Deploy
mkdir -p ~/Applications/Pill.app/Contents/MacOS
cp .build/debug/Pill ~/Applications/Pill.app/Contents/MacOS/Pill
cp Sources/DynamicNotch/Info.plist ~/Applications/Pill.app/Contents/Info.plist
open ~/Applications/Pill.app
```

**Requirements:** macOS 14.0+ · Apple Silicon (arm64) · Swift 5.9+
